> 4. Open `config.cfg` in your favorite text editor. Specify the users you wish to create in the `users` list. 
> Don't forget to escape nicknames, if needed. e.g. for leading zero in nicknames: 
```
users:
  - '001'
  - '002'
  - '003' 
  ```
